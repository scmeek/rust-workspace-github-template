# Code of Conduct

## Our Commitment

As the maintainer and contributors of this project, we are committed to creating a welcoming, respectful, and collaborative environment for everyone who wants to participate. We value contributions from people of all backgrounds and experience levels.

## Our Standards

We encourage behavior that contributes to a positive and productive environment:

- Being respectful and considerate in communication
- Welcoming newcomers and helping them get started
- Being open to different viewpoints and experiences
- Accepting constructive criticism gracefully
- Focusing on what's best for the project and community
- Showing empathy towards others

We do not tolerate behavior that creates a hostile or uncomfortable environment:

- Harassment, intimidation, or discrimination of any kind
- Insulting, derogatory, or inflammatory comments
- Personal or political attacks
- Publishing others' private information without permission
- Trolling or sustained disruption of discussions
- Other conduct that would be inappropriate in a professional setting

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub repositories (issues, pull requests, discussions)
- Project documentation and wikis
- Communication channels
- Any space where someone is representing the project

## Enforcement

If you experience or witness behavior that violates this Code of Conduct, please report it by:

- Opening a confidential issue
- Using GitHub's report feature for serious violations

All reports will be handled with discretion

### Response to Violations

Depending on the severity and circumstances, responses may include:

- A private conversation to clarify expectations
- A public or private warning
- Temporary restriction from project participation
- Permanent removal from the project

## Attribution

This Code of Conduct is inspired by the Contributor Covenant and other community guidelines.

## Questions

If you have questions about this Code of Conduct or want to suggest improvements, please open an issue or discussion.
